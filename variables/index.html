Variables rules

в программировании есть только две сложные вещи: инвалидация кеша, выбор имени переменной, и ошибки на единицу.

Boolean

ch-var-warning-bool-part-of-speech

Название для булевой переменной должно быть общим вопросом (ответом на общий вопрос может быть либо да, либо нет).

Используй в названиях такие частицы как is, has, was и т.д.

Плохо

# Открыто то ли модальное окно
open = False

# Есть ли у пользователя доступ
access = False

// Открыто то ли модальное окно
const open = false

// Есть ли у пользователя доступ
const access = false

Хорошо

# Открыто то ли модальное окно
is_open = False

# Есть ли у пользователя доступ
has_access = False

// Открыто то ли модальное окно
const isOpen = false

// Есть ли у пользователя доступ
const hasAccess = false
